## Recommendations

- Use PNG format with transparent backgrounds
- Recommended height: 80-120px (width will scale automatically)
- Ensure you have permission to use the institutional logos
- Consider using official logos from each institution's brand guidelines:
  - UW: https://www.washington.edu/brand/graphic-elements/primary-logos/
  - PNNL: Contact their communications office
  - Cedars-Sinai: Contact their communications office
  - Buffalo: https://www.buffalo.edu/brand.html

## Fallback Behavior

If logo files are not present, the website will display text placeholders with the institution name instead. This ensures the site works even without the actual logo images.
